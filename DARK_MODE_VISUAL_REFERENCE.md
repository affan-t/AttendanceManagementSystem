## ?? User Interaction Flow

### Step-by-Step Toggle Process

```
1. User sees interface in current mode
   ?
2. User clicks toggle button (header or sidebar)
   ?
3. JavaScript captures click event
   ?
4. Current theme detected (light or dark)
   ?
5. Theme switched (dark ? light)
   ?
6. data-theme attribute updated on <html>
   ?
7. CSS variables recalculate instantly
   ?
8. Smooth 300ms transition begins
   ?
9. All colors fade to new values
   ?
10. New theme fully applied
    ?
11. Preference saved to localStorage
    ?
12. Icon/text updated (moon ? sun)
    ?
13. Console logs "Theme changed to: [theme]"
```
